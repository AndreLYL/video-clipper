# VideoClipper v1.1.0 发布说明

## 🎉 新增功能

### 批量裁剪增强

1. **HTML报告自动生成** 📊
   - 批量裁剪完成后自动生成精美的HTML报告
   - 报告包含裁剪任务的完整统计信息（总计、成功、失败）
   - 每个裁剪任务显示详细信息和状态

2. **视频帧预览** 🎬
   - 每个成功裁剪的视频自动提取首帧和尾帧
   - 帧图像以base64格式嵌入HTML，无需额外文件
   - 图像自动缩放优化（最大宽度400px）

3. **图片放大查看** 🔍
   - 点击任意帧图像即可全屏查看
   - 平滑的淡入淡出和缩放动画效果
   - 显示详细的片段信息和帧标题
   - 支持多种关闭方式：
     - 点击关闭按钮（×）
     - 点击背景区域
     - 按ESC键

4. **多种时间格式支持** ⏰
   - `HH:MM:SS` - 标准格式（如：12:30:45）
   - `YYYY-MM-DD HH:MM:SS` - 日期时间格式（如：2025-11-13 00:26:39）
   - `YYYY年MM月DD日HH:MM:SS` - 中文日期格式（如：2025年11月13日00:26:50）
   - `HH点MM分SS秒` - 中文时间格式（如：00点34分20秒）

### 报告功能

5. **现代化报告设计** 🎨
   - 渐变紫色主题，视觉效果出众
   - 卡片式布局，信息层次清晰
   - 悬停动画效果
   - 响应式设计，支持移动设备

6. **智能报告管理** 📁
   - HTML报告保存在视频目录中
   - 报告文件名带时间戳（裁剪报告_YYYYMMDD_HHMMSS.html）
   - 完成后询问是否在浏览器中打开

### 用户体验优化

7. **交互体验提升** ✨
   - 图片悬停时轻微放大效果
   - 鼠标指针变化提示可点击
   - 实时进度显示（裁剪、提取帧、生成报告）
   - 详细的错误信息反馈

## 🔧 技术改进

- 新增 PIL/Pillow 图像处理
- Base64 图像编码优化
- 线程安全的UI更新
- 增强的错误处理机制
- 优化的视频帧提取算法

## 📝 使用说明

### 批量裁剪模式

1. 选择批量裁剪模式
2. 选择源视频和目标目录
3. 输入视频起始时间（支持多种格式）
4. 选择时间戳文件（支持多种时间格式）
5. 点击开始裁剪
6. 完成后查看HTML报告
7. 点击帧图像可放大查看

### 时间戳文件示例

```
# 批量裁剪时间戳文件
# 支持多种时间格式

12:30:45 第一个片段
2025-11-13 00:26:39 第二个片段
2025年11月13日00:26:50 第三个片段
00点34分20秒 第四个片段
```
