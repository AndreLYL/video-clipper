# 视频裁剪工具 v1.0.0 发布说明

**发布日期**: 2025年11月10日  
**作者**: andre.li

## 🎉 首个正式版本发布

这是视频裁剪工具的首个正式版本，提供了简洁易用的视频裁剪功能，支持单点裁剪和批量裁剪两种模式。

## ✨ 主要功能

### 1. 单点裁剪模式
- 输入视频起始UTC时间（格式：HH:MM:SS）
- 可配置向前和向后裁剪的秒数（默认：向前40秒，向后20秒）
- 输入要裁剪的时间点
- 自动生成以时间戳命名的视频片段

### 2. 批量裁剪模式
- 支持从文本文件中读取多个时间点
- 文件格式：每行一个时间点，格式为 `HH:MM:SS 描述`
- 支持 `#` 开头的注释行
- 自动处理多个时间点，一次性完成批量裁剪

### 3. 灵活的裁剪时长配置
- 向前裁剪秒数：可自定义（默认40秒）
- 向后裁剪秒数：可自定义（默认20秒）
- 适应不同的视频裁剪需求

### 4. 智能错误检测
- 自动检测视频时长
- 验证裁剪时间是否在有效范围内
- 提供详细的错误提示和建议
- 批量裁剪时自动跳过无效时间点，继续处理有效项

## 🎨 用户界面

- **现代化深色主题**：采用舒适的深色配色方案
- **卡片式布局**：清晰的功能分区
- **实时进度显示**：可视化的处理状态和进度条
- **响应式设计**：适应不同窗口尺寸
- **悬停效果**：提升交互体验

## 💻 技术特性

- **多线程处理**：视频裁剪在后台线程执行，不阻塞UI
- **资源管理**：自动释放视频资源，避免内存泄漏
- **线程安全**：UI更新采用线程安全的方式
- **格式支持**：支持常见视频格式（MP4、AVI、MOV、MKV、FLV等）

## 📦 文件说明

- **VideoClipper_v1.0.0.exe**: 主程序（单文件EXE，无需安装，约87MB）
- **timestamp_example.txt**: 批量裁剪时间戳文件示例

## 🚀 使用方法

1. 双击运行 `VideoClipper_v1.0.0.exe`
2. 选择源视频文件
3. 选择输出目录
4. 选择裁剪模式（单点裁剪或批量裁剪）
5. 输入视频起始UTC时间
6. 配置向前/向后裁剪秒数（可选）
7. 根据模式输入裁剪时间点或选择时间戳文件
8. 点击"开始裁剪"按钮

## 📝 时间戳文件格式

批量裁剪模式下的时间戳文件格式示例：

```
# 时间戳文件示例
# 格式: HH:MM:SS 描述
# 每行一个时间点，描述可选
# 以#开头的行为注释

12:30:45 第一个片段
14:25:30 第二个片段
16:45:20 第三个片段
```

## ⚠️ 注意事项

- 裁剪时长必须是正整数
- 确保视频时长足够进行裁剪
- 裁剪时间不能超出视频范围
- 处理过程中请勿关闭程序
- 首次运行可能需要较长的加载时间（初始化视频处理库）
