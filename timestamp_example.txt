# 视频裁剪时间戳文件示例
# 支持多种时间格式，每行一个时间点（可选添加描述）
# 支持 # 开头的注释行和空行
# ⚠ 注意: 必须包含秒数，否则将跳过该条目

# 格式1: HH:MM:SS (时:分:秒)
12:30:45 第一个片段
00:15:30 第二个片段

# 格式2: YYYY-MM-DD HH:MM:SS (完整日期时间)
2025-11-13 00:26:39 日期格式片段

# 格式3: YYYY年MM月DD日HH:MM:SS (中文日期完整时间)
2025年11月13日00:26:50 中文日期片段

# 格式4: HH点MM分SS秒 (中文时间)
00点34分20秒 中文时间片段

# 兼容性: 支持中文标点符号（自动转换）
12：35：00 中文冒号（全角）

# 混合格式示例
2025-11-13 12：35：00 混合标点符号

# ⚠ 以下格式会被跳过（缺少秒数）：
# 12:30 (缺少秒数，会被跳过)
# 12点13分 (缺少秒数，会被跳过)
# 2025-11-13 12:35 (缺少秒数，会被跳过)
