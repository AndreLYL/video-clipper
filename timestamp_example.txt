# 视频裁剪时间戳文件示例
# 支持多种时间格式，每行一个时间点（可选添加描述）
# 支持 # 开头的注释行和空行

# 格式1: HH:MM:SS (时:分:秒)
12:30:45 第一个片段
00:15:30 第二个片段

# 格式2: HH:MM (时:分，秒数自动使用"向前裁剪秒数")
12:37 简化格式片段（如向前40秒，则实际为12:37:40）
00:15 另一个简化片段

# 格式3: YYYY-MM-DD HH:MM:SS (完整日期时间)
2025-11-13 00:26:39 日期格式片段

# 格式4: YYYY-MM-DD HH:MM (日期+简化时间)
2025-11-13 12:35 日期简化片段

# 格式5: YYYY年MM月DD日HH:MM:SS (中文日期完整时间)
2025年11月13日00:26:50 中文日期片段

# 格式6: YYYY年MM月DD日HH:MM (中文日期简化时间)
2025年11月13日12:35 中文日期简化片段

# 格式7: HH点MM分SS秒 (中文时间)
00点34分20秒 中文时间片段

# 格式8: HH点MM分 (中文简化时间)
12点13分 中文简化片段

# 兼容性: 支持中文标点符号（自动转换）
12：35：00 中文冒号（全角）
12：30 中文冒号简化

# 混合格式示例
2025-11-13 12：35 混合标点符号

# 说明：
# - 缺少秒数时，将自动使用"向前裁剪秒数"作为默认值
# - 例如：向前40秒，向后20秒的配置下
#   12:37 会被解析为 12:37:40
#   然后裁剪 12:36:40 到 12:38:00 的视频（共60秒）
