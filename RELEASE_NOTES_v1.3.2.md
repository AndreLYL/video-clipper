# 视频裁剪工具 v1.3.2 发布说明

## 🎯 主要更新

### 🔒 稳定性与兼容性
1. **自动修复 MoviePy 无法读取时长的问题**
   - 新增 `ensure_video_readable()` 流程，先尝试直接读取视频时长
   - 解析失败时自动调用 FFmpeg 使用 `-fflags +genpts` / `-c copy` 重封装生成临时文件
   - 修复后的临时视频会继续沿用同一裁剪流程，用户无须干预

2. **临时文件安全清理**
   - 为所有临时重封装目录建立集中管理
   - 每次裁剪（单个/批量）结束后自动清理，避免磁盘空间被占用

### 📝 文档与流程
1. **README**：更新为 v1.3.2，强调新增的自动修复能力与最新下载路径  
2. **项目结构 / Release 指南**：同步版本号、下载链接与操作示例  
3. **版本信息**：`VideoClipper.spec`、`file_version_info.txt`、UI 标识全部更新为 v1.3.2

## 🧪 建议验证
- 使用曾经触发 “failed to read the duration” 的原始视频进行回归测试
- 观察 FFmpeg 自动修复提示后继续裁剪流程是否顺畅
- 确认裁剪完成后 `temp` 目录未残留

## 📦 系统要求

- Windows 10/11
- 无需安装 Python 或其他依赖

## 🚀 下载

请下载 `VideoClipper_v1.3.2.exe` 文件（GitHub Release 资源），双击即可运行。
