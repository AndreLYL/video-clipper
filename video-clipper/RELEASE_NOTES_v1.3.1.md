# 视频裁剪工具 v1.3.1 发布说明

## 🎯 主要更新

### 🐛 Bug修复

1. **修复时间戳解析问题**
   - 修复了批量裁剪模式下，时间和描述之间没有空格分隔时解析失败的问题
   - 现在支持所有时间格式的时间和描述之间有无空格分隔
   - 例如：`2025-11-18 15:37:15红绿灯识别错误` 现在可以正常解析

### ✨ 功能改进

1. **更灵活的时间戳格式**
   - 支持 `2025-11-18 15:37:15 描述`（有空格）
   - 支持 `2025-11-18 15:37:15描述`（无空格）
   - 支持 `12:30:45 描述`（有空格）
   - 支持 `12:30:45描述`（无空格）
   - 支持所有时间格式的灵活分隔

2. **项目结构优化**
   - 项目已整理到 `video-clipper/` 文件夹
   - 更清晰的文件组织结构
   - 独立的Git配置

## 📝 使用示例

### 时间戳文件格式

现在支持以下所有格式：

```
# 有空格分隔
2025-11-18 15:37:15 红绿灯识别错误
12:30:45 第一个片段

# 无空格分隔（新支持）
2025-11-18 15:37:15红绿灯识别错误
12:30:45第一个片段

# 中文格式
2025年11月18日15:37:15红绿灯识别错误
12点30分45秒第一个片段
```

## 🔧 技术细节

- 使用正则表达式 `\s*` 匹配0个或多个空格
- 支持所有时间格式的灵活分隔
- 向后兼容，不影响现有功能

## 📦 系统要求

- Windows 10/11
- 无需安装Python或其他依赖
