## 📝 简介

视频裁剪工具是一款功能强大、简洁易用的视频片段裁剪软件，支持单点裁剪和批量裁剪两种模式，采用现代化深色主题UI设计。v1.3.2 对时间解析与视频容错能力进行了全面升级：自动修复无法读取时长的视频，并继续沿用HTML报告、帧预览、图片放大等高级特性。

## ✨ 主要特性

- 🎬 **单点裁剪**：精确裁剪单个视频片段
- 📋 **批量裁剪**：从文本文件读取多个时间点，一次性完成批量裁剪
- 📊 **HTML报告**：自动生成包含统计信息和视频帧预览的精美报告
- 🖼️ **帧预览**：每个裁剪视频显示首帧和尾帧，支持点击放大查看
- ⏰ **多格式支持**：支持多种时间格式（HH:MM:SS、日期时间、中文格式等）
- ⚙️ **灵活配置**：可自定义向前/向后裁剪秒数（默认：前40秒，后20秒）
- 🎨 **现代化UI**：深色主题，卡片式布局，良好的用户体验
- 🚀 **智能检测**：自动验证时间范围，提供详细错误提示
- 💻 **单文件EXE**：无需安装，双击即用

## 🚀 快速开始

### 1. 启动程序

双击运行 `VideoClipper_v1.3.2.exe`

### 2. 单点裁剪模式

1. 点击"浏览"选择源视频文件
2. 选择保存目录
3. 选择"单点裁剪"模式
4. 输入视频起始UTC时间（格式：HH:MM:SS，例如：12:00:00）
5. 配置向前/向后裁剪秒数（可选，默认40/20秒）
6. 输入要裁剪的时间点（格式：HH:MM:SS，例如：12:30:45）
7. 点击"开始裁剪"

**示例**：
- 视频起始时间：`12:00:00`
- 裁剪时间点：`12:30:45`
- 向前40秒，向后20秒
- 结果：裁剪出从12:30:05到12:31:05的60秒视频片段

### 3. 批量裁剪模式

1. 点击"浏览"选择源视频文件
2. 选择保存目录
3. 选择"批量裁剪"模式
4. 输入视频起始UTC时间（格式：HH:MM:SS）
5. 配置向前/向后裁剪秒数（可选）
6. 点击"选择文件"，选择时间戳文本文件
7. 点击"开始裁剪"

#### 时间戳文件格式

创建一个文本文件（如 `timestamps.txt`），每行一个时间点。

**v1.1.0新增：支持多种时间格式！**

```
# 时间戳文件示例
# 支持多种时间格式
# 以#开头的行为注释

# 格式1: HH:MM:SS
12:30:45 第一个片段

# 格式2: YYYY-MM-DD HH:MM:SS
2025-11-13 00:26:39 第二个片段

# 格式3: YYYY年MM月DD日HH:MM:SS
2025年11月13日00:26:50 第三个片段

# 格式4: HH点MM分SS秒
00点34分20秒 第四个片段
```

参考示例文件：`timestamp_example.txt` 或 `timestamp_test.txt`

**批量裁剪完成后会自动生成HTML报告，包含：**
- 📊 裁剪统计信息（总计、成功、失败）
- 🎬 每个视频的首帧和尾帧预览
- 🔍 点击图片可放大查看
- 📁 报告保存在视频目录中

## 📖 使用说明

### 时间格式

**单点裁剪模式**：使用 `HH:MM:SS` 格式
- `HH`: 小时（00-23）
- `MM`: 分钟（00-59）
- `SS`: 秒（00-59）

**批量裁剪模式（v1.1.0新增）**：支持多种格式
1. `HH:MM:SS` - 标准格式（如：12:30:45）
2. `YYYY-MM-DD HH:MM:SS` - 日期时间格式（如：2025-11-13 00:26:39）
3. `YYYY年MM月DD日HH:MM:SS` - 中文日期格式（如：2025年11月13日00:26:50）
4. `HH点MM分SS秒` - 中文时间格式（如：00点34分20秒）

### 裁剪时长配置

- **向前秒数**：从指定时间点向前裁剪的秒数
- **向后秒数**：从指定时间点向后裁剪的秒数
- 例如：时间点 12:30:00，向前40秒，向后20秒
  - 裁剪范围：12:29:20 - 12:30:20（共60秒）

### 输出文件命名

- 单点裁剪：`HH-MM-SS.mp4`（例如：`12-30-45.mp4`）
- 批量裁剪：`HH-MM-SS_描述.mp4`（例如：`12-30-45_第一个片段.mp4`）

## ⚠️ 注意事项

1. **视频时长验证**：
   - 确保视频时长足够进行裁剪
   - 裁剪时间不能超出视频范围
   - 程序会自动检测并提示错误

2. **输入验证**：
   - 时间格式必须严格遵守 HH:MM:SS
   - 裁剪时长必须是正整数
   - 裁剪时间点不能早于视频起始时间

3. **处理过程**：
   - 处理过程中请勿关闭程序
   - 大文件处理可能需要较长时间
   - 进度条会显示当前处理状态

4. **首次运行**：
   - 首次运行可能需要10-30秒加载时间
   - 这是正常现象，请耐心等待

## 🐛 常见问题

### Q: 程序无法启动？
A: 
- 检查是否被杀毒软件拦截
- 确保系统是Windows 10/11
- 尝试以管理员身份运行

### Q: 裁剪失败？
A: 
- 检查视频文件是否完整
- 确认时间范围是否在视频时长内
- 查看错误提示信息

### Q: 批量裁剪部分失败？
A: 
- 程序会跳过无效时间点，继续处理其他片段
- 查看完成后的错误报告
- 检查时间戳文件格式是否正确
